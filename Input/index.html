Input

Inputs are necessary for user interaction with the website. Inputs are used to take information from the user, and then save the data to the server, accordingly.

Error Input

Error input a state where the value it contains has an error(errors)

Success Input

Success input a state where the value it contains is valid

Text area

The HTML textarea tag is used to make input field with multiple lines in a form and can hold an unlimited number of character
